# K&P — аренда PlayStation 5

Адаптивный одностраничный сайт для сервиса аренды игровых консолей в Омске.

## Запуск

Откройте `index.html` в браузере или запустите в папке любой статический сервер.

Сайт не использует сборщик и внешние JavaScript-зависимости. В нём есть адаптивная вёрстка, форма заявки с понятным подтверждением, FAQ и плавная навигация по разделам.
