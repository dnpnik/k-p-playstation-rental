# K&P — аренда PlayStation 5

Адаптивный четырёхстраничный сайт для сервиса аренды игровых консолей в Омске.

- `index.html` — главная;
- `sets.html` — комплекты;
- `service.html` — сервис и процесс;
- `contacts.html` — контакты и FAQ.

## Запуск

Откройте `index.html` в браузере или запустите в папке любой статический сервер.

Сайт не использует сборщик и внешние JavaScript-зависимости. В нём есть адаптивная вёрстка, динамический видеопрогресс, фотокарточки комплектов, мобильная панель быстрого заказа, доступное меню, FAQ и плавная навигация по разделам.

В `assets/media/` находятся оптимизированные WebP-фотографии, новый студийный product-shot консоли и короткий MP4-ролик для живых медиасекций. Ролик запускается без звука, поддерживает паузу и учитывает системную настройку уменьшения движения.

Форма заявки работает в демонстрационном режиме и честно сообщает, что данные не отправляются, пока не подключён реальный канал приёма заявок.
